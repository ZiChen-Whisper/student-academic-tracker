### 如何将数据集转为 SQL 文件

以下提供三种常用方法，任选其一即可：

**方法一：使用 Python（推荐）**

```python
import pandas as pd
from sqlalchemy import create_engine, text

# 1. 读取数据集
df = pd.read_csv("StudentPerformance.csv")  # 或 pd.read_excel(...)
# df = pd.read_excel("student_performance.xlsx")  # excel 文件

# 2. 创建 SQLite 内存数据库，写入数据
engine = create_engine("sqlite:///:memory:")
df.to_sql("student_performance", engine, index=False, if_exists="replace")

# 3. 导出为 SQL 文件
with engine.connect() as conn:
    result = conn.execute(text("SELECT * FROM student_performance"))
    columns = result.keys()
    
    # 注意：去掉引号内多余空格
    with open("Zichen_Zou_student_performance.sql", "w", encoding="utf-8") as f:
        # 写入建表语句
        ddl = pd.io.sql.get_schema(df, "student_performance", con=engine)
        f.write(ddl + ";\n\n")
        
        # 写入插入语句
        for _, row in df.iterrows():
            values = ", ".join([f"'{v}'" if isinstance(v, str) else str(v) for v in row])
            f.write(f"INSERT INTO student_performance ({', '.join(columns)}) VALUES ({values});\n")
```

> 需要安装依赖：`pip install pandas sqlalchemy`
>
> 已经改过了，这个版本的脚本可以用，使用方法就是把csv或者excel格式的文件放在和脚本同一个目录下直接运行脚本。

**方法二：使用在线工具**

1. 打开 [ConvertCSV](https://www.convertcsv.com/csv-to-sql.htm) 或 [SQLizer](https://sqlizer.io/)
2. 上传你的 csv/excel 文件
3. 设置表名（用英文，如 `student_performance`）
4. 点击转换，下载生成的 SQL 文件

**方法三：使用数据库客户端**

1. 用 Navicat / DBeaver / DataGrip 等工具连接任意数据库
2. 导入 csv/excel 文件到一张表
3. 右键该表 → 导出 / 转储 → 选择"仅结构和数据" → 导出为 `.sql` 文件

> ⚠️ 无论用哪种方法，请确保生成的 SQL 文件同时包含 **CREATE TABLE** 和 **INSERT** 语句。
