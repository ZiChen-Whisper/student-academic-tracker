# Markdown 简明教程

> 本教程帮助你快速掌握 Markdown 常用语法，轻松编写格式化文档。

---

## 一、标题

在行首加 `#` 号表示标题，`#` 越多标题越小：

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

**效果：**

# 一级标题
## 二级标题
### 三级标题
#### 四级标题

---

## 二、文字样式

| 语法 | 效果 | 说明 |
|------|------|------|
| `**加粗文字**` | **加粗文字** | 双星号包裹 |
| `*斜体文字*` | *斜体文字* | 单星号包裹 |
| `***加粗斜体***` | ***加粗斜体*** | 三星号包裹 |
| `~~删除线~~` | ~~删除线~~ | 双波浪线包裹 |

---

## 三、列表

### 无序列表

用 `-` 或 `*` 开头：

```markdown
- 苹果
- 香蕉
- 橘子
```

**效果：**

- 苹果
- 香蕉
- 橘子

### 有序列表

用数字加点开头：

```markdown
1. 第一步
2. 第二步
3. 第三步
```

**效果：**

1. 第一步
2. 第二步
3. 第三步

### 嵌套列表

缩进两个空格即可嵌套：

```markdown
- 水果
  - 苹果
  - 香蕉
- 蔬菜
  - 西红柿
  - 黄瓜
```

**效果：**

- 水果
  - 苹果
  - 香蕉
- 蔬菜
  - 西红柿
  - 黄瓜

## 四、链接与图片

### 链接

```markdown
[显示文字](网址)
```

示例：`[百度](https://www.baidu.com)` → [百度](https://www.baidu.com)

### 图片

和链接类似，前面多一个 `!`：

```markdown
![图片描述](图片地址)
```

示例：`![示例图片](https://via.placeholder.com/150)` → ![示例图片](https://via.placeholder.com/150)

---

## 五、引用

用 `>` 表示引用：

```markdown
> 这是一段引用文字。
> 可以有多行。
```

**效果：**

> 这是一段引用文字。
> 可以有多行。

---

## 六、代码

### 行内代码

用反引号 `` ` `` 包裹：`print("Hello")` → `print("Hello")`

### 代码块

用三个反引号包裹，并可指定语言：

````markdown
```python
def hello():
    print("Hello, World!")
```
````

**效果：**

```python
def hello():
    print("Hello, World!")
```

---

## 七、表格

用 `|` 分隔列，用 `-` 分隔表头和内容：

```markdown
| 姓名 | 年龄 | 城市 |
|------|------|------|
| 张三 | 25   | 北京 |
| 李四 | 30   | 上海 |
```

**效果：**

| 姓名 | 年龄 | 城市 |
|------|------|------|
| 张三 | 25   | 北京 |
| 李四 | 30   | 上海 |

> **提示**：对齐方式可在 `-` 行中用 `:` 控制：
> - `:---` 左对齐（默认）
> - `---:` 右对齐
> - `:---:` 居中对齐

## 九、任务列表（部分编辑器支持）

```markdown
- [x] 已完成的任务
- [ ] 未完成的任务
```

**效果：**

- [x] 已完成的任务
- [ ] 未完成的任务

## 十、转义字符

如果需要显示特殊符号本身，在前面加 `\`：

| 写法 | 效果 |
|------|------|
| `\*` | \* |
| `\#` | \# |
| `\-` | \- |
| `\|` | \| |

## 速查表

| 想要的效果 | 怎么写 |
|-----------|--------|
| **加粗** | `**文字**` |
| *斜体* | `*文字*` |
| ~~删除线~~ | `~~文字~~` |
| [链接](url) | `[文字](url)` |
| ![图片](url) | `![描述](url)` |
| 行内代码 | `` `代码` `` |
| 引用 | `> 文字` |
| 无序列表 | `- 文字` |
| 有序列表 | `1. 文字` |
| 分割线 | `---` |
| 表格 | `\| 表头 \|` |
